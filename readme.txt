== Description ==
WP Conditional Media Folder automatically saves uploaded media files to custom server directories based on configurable filename or MIME type rules while ensuring full integration and accessibility within the WordPress Media Library.
Caratteristiche:
* Custom Storage Path: Save files anywhere on the server filesystem (absolute path).
* Rules Engine (OR/AND): Create multiple set of criteria.
* Avaiable Conditions: Starts with, Ends with, Contains, Length Min/Max, MIME type.
* Seamless Integration: Files appear in the Media Library regardless of storage location.
* Internationalization: Translations ready (English and Italian included).
== Installation ==
1. Load wp-conditional-media-folder into /wp-content/plugins/ directory.
2. Activate the plugin via 'Plugins' menu.
3. Go to Settings > Conditional Media Folder.
4. Insert destination path absolute path and public URL.
5. Add rules to recognize the files.
